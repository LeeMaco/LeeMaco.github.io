# 會員管理系統

一個功能完整的現代化會員管理系統，採用純前端技術構建，支持離線使用、照片處理、資料導入導出等豐富功能。

## 🌟 主要功能

### 📋 會員資料管理
- **新增會員**：支持基本資料輸入和照片上傳
- **編輯會員**：修改會員資料和照片
- **刪除會員**：安全刪除會員資料和相關照片
- **查看會員**：顯示會員詳細資料和完整記錄

### 📸 照片處理
- **照片上傳與預覽**：支持多種圖片格式
- **照片編輯**：內建編輯器支持縮放、旋轉、裁剪
- **照片優化**：自動壓縮和格式優化

### 📊 資料導入導出
- **多格式支持**：JSON、Excel (CSV) 格式
- **批量操作**：支持大量資料的快速導入
- **選擇性導出**：可選擇是否包含照片資料
- **自動備份**：定期自動備份重要資料

### 🔍 搜尋與過濾
- **即時搜尋**：支持姓名、電話號碼搜尋
- **智能過濾**：快速定位目標會員
- **搜尋歷史**：記住常用搜尋條件

### 💾 資料存儲與備份
- **本地存儲**：使用 IndexedDB 安全存儲資料
- **離線支持**：完全離線可用，無需網路連接
- **自動備份**：定期創建資料備份
- **存儲監控**：實時監控存儲空間使用情況

### 📱 使用者體驗優化
- **響應式設計**：完美適配手機、平板、桌面
- **PWA 支持**：可安裝到主屏幕，如原生應用
- **操作提示**：友好的用戶界面和操作指引
- **確認對話框**：重要操作前的安全確認

## 🚀 快速開始

### 系統要求
- 現代瀏覽器（Chrome 80+、Firefox 75+、Safari 13+、Edge 80+）
- 支持 IndexedDB 和 Service Worker
- 建議至少 100MB 可用存儲空間

### 安裝使用

1. **直接使用**
   ```bash
   # 克隆或下載項目文件
   # 使用本地服務器打開 index.html
   python3 -m http.server 8000
   # 或使用 Node.js
   npx serve .
   ```

2. **安裝為 PWA**
   - 在支持的瀏覽器中打開應用
   - 點擊地址欄的「安裝」按鈕
   - 或通過瀏覽器菜單選擇「添加到主屏幕」

## 📖 使用指南

### 新增會員
1. 點擊「新增會員」按鈕
2. 填寫必要資訊（姓名、電話）
3. 可選擇上傳會員照片
4. 點擊「儲存」完成新增

### 編輯照片
1. 在會員表單中上傳照片
2. 點擊「編輯」按鈕打開照片編輯器
3. 使用滑桿調整縮放和旋轉
4. 點擊「裁剪」或「確認」保存修改

### 資料導出
1. 點擊「導出」按鈕
2. 選擇導出格式（JSON 或 Excel）
3. 選擇是否包含照片資料
4. 點擊「開始導出」下載檔案

### 資料導入
1. 點擊「導入」按鈕
2. 選擇要導入的檔案（JSON 或 Excel）
3. 點擊「開始導入」處理資料
4. 系統會自動驗證和導入有效資料

### 搜尋會員
1. 在搜尋框中輸入姓名或電話
2. 系統會即時顯示匹配結果
3. 點擊會員卡片查看詳細資訊

## 🛠️ 技術架構

### 前端技術
- **HTML5**：語義化標記和現代 Web API
- **CSS3**：響應式設計、Grid 布局、動畫效果
- **JavaScript ES6+**：模組化、異步處理、類別語法

### 存儲技術
- **IndexedDB**：客戶端資料庫存儲
- **Local Storage**：設定和偏好存儲
- **Cache API**：資源緩存和離線支持

### PWA 功能
- **Service Worker**：離線緩存、背景同步
- **Web App Manifest**：應用安裝和主屏幕圖標
- **Push Notifications**：推送通知支持

### 照片處理
- **Canvas API**：圖片編輯和處理
- **File API**：檔案上傳和讀取
- **Blob API**：二進制資料處理

## 🔒 隱私與安全

- **本地存儲**：所有資料僅存儲在用戶設備上
- **無服務器**：不依賴外部服務器，保護隱私
- **加密支持**：可選擇啟用本地資料加密
- **安全刪除**：徹底清除已刪除的資料

## 🐛 故障排除

### 常見問題

**Q: 照片上傳失敗**
A: 檢查圖片格式（支持 JPG、PNG、GIF）和大小（限制 5MB）

**Q: 資料導入失敗**
A: 確認檔案格式正確，JSON 檔案需包含有效的會員資料陣列

**Q: 應用無法離線使用**
A: 確保瀏覽器支持 Service Worker，首次訪問需要網路連接

**Q: 存儲空間不足**
A: 清理不需要的會員資料或照片，或使用資料導出功能備份後清理
